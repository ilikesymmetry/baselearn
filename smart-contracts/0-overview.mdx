## Goals
After this section, you should be able to:
1. Read and understand Solidity smart contracts
1. Interact with Solidity smart contracts via CLI and block explorers (e.g. [BaseScan](https://basescan.org/))
1. Write and deploy Solidity smart contracts

## Setup

Before beginning, you'll need to setup your environment.
1. Install [Coinbase Wallet](https://chromewebstore.google.com/detail/coinbase-wallet-extension/hnfanknocfeofbddgcijnmhnfnkdnaad) chrome extension and create a wallet if you don't have one already
1. Open [P2P Faucet](https://p2p-faucet.vercel.app/) and connect your wallet to acquire Base Testnet ("Base Sepolia") ETH
1. Install [Foundry](https://book.getfoundry.sh/getting-started/installation)
1. Import your Coinbase Wallet into Foundry
    1. Copy your private key from Coinbase Wallet extension settings
    1. Run `cast wallet import [ACCOUNT_NAME] --private-key [PRIVATE_KEY]`
    1. Enter password into prompt to encrypt your private key
